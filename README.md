## **Getting Started**

1. **Clone the Repository**:
   ```bash
   https://github.com/Nipun2001M/Flutter-Jokes-App.git
   ```

2. **Navigate to the Directory**:
   ```bash
   cd Flutter-Jokes-App
   ```

3. **Fetch Dependencies**:
   ```bash
   flutter pub get
   ```

4. **Run the App**:
   ```bash
   flutter run
   ```
